Banco de datos

Conjunto de datos relacionados que constituyen una fuente de información a disposición de numerosos usuarios (AF&AL, 1996).

Baremo

Tabla que relaciona las puntuaciones directas con puntuaciones normalizadas. Tabla de anotaciones previa a la evaluación de los resultados donde se indica el valor de cada respuesta posible (AL&AF, 1996).

Base de datos

Conjunto de registros organizados en campos que contienen información, almacenados en soporte óptico o magnético, accesible por ordenados (AF&AL, 1996).

Base de datos factual

Base de datos que incluye la totalidad de un documento textual, numérico o icónico (AF&AL, 1996).

Base de datos referencial

Base de datos que informa o describe documentos externos a la base. Puede ser bibliográfica, de directorios o de recursos (AF&AL, 1996).

Base de tests

Combinación de tests con la intención de incrementar su validez en la medida de la característica a medir (AF&AL, 1996).

Bibliografía

Ciencia auxiliar que mediante un conjunto de técnicas de identificación y descripción de los documentos facilita el conocimiento científico de una disciplina. Relación de textos (libros, manuales, artículos, etc.) relevantes sobre un tema determinado (AF&AL, 1996).

Bibliografía gris

Bibliografía que no sigue los canales normales de publicación o edición, constituida normalmente por cartas, apuntes, anotaciones, etc. (AF&AL,1996).

Biografía

Conjunto de relatos orales (transcritos) y escritos que junto con otras fuentes documentales constituyen el itinerario de la vida de una persona (AF&AL, 1996).

Bivariado

Relativo a dos variables (AF&AL, 1996).

Bloque

Grupo de personas que constituye un conjunto homogéneo respecto a una variable, llamada variable de bloqueo (AF&AL, 1996).

Bloqueo

Fijación de una variable en un valor fijo y conocido (AF&AL, 1996).

Brainstorming

Técnica de discusión que favorece las ocurrencias creativas de un grupo que trata de hallar soluciones a un problema reuniendo iniciativas espontáneas que más tarde se analizarán y someterán a crítica (AF&AL, 1996).

Búsqueda

Examen de los elementos de una base de datos para hallar aquellos que tienen una determinada propiedad o condición (AF&AL, 1996).
